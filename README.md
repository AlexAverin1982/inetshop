# Домашняя работа "17.1 Исключения. Пользовательские исключения"
## Цель домашней работы
Используя навыки, полученные в ходе обучения, улучшить объектную модель, 
применив механизм вызова исключений для контроля ошибок 
и адекватного информирования пользователя системы; разработать класс пользовательского исключения,
применяемого только в предметной области приложения в целях более точного контроля за ошибками.
Новый функционал подразумевает:
- Информирование пользователя о возникающих ошибках и предотвращение останова работы приложения ввиду 
  возникновения некритических проблем. 

## Возможности приложения
+ Считывание данных о товарах и категориях из файла в формате json

## Структура проекта
#### + Исходный код функционала приложения находится в папке `src`
  + модуль `main` - точка входа приложения, отображает результаты загрузки данных из файла
  + модуль `classes` - реализация объектной модели, классы "Товар" и "Категория товаров" с начальным набором свойств 
  и методов, а так же объектная модель пользовательских классов исключений.
#### + Тесты работы исходного кода находится в папке `tests`
#### + Данные, с которыми будет работать приложение находятся в папке `data`
#### + Данные отчета о покрытии кода тестами находятся в папке htmlcov
Основновная информация доступна через файл `index.html`



## Инструкции по использованию приложения
- Запустите модуль `src\main.py`, следуйте инструкциям на экране
- Отдельными компонентами можно пользоваться, импортируя их из соответствующих модулей папки `src`
